# Laporan Evaluasi Singkat

Hasil pengembangan telah memenuhi kebutuhan utama pada soal:

- Aplikasi berjalan di localhost.
- Mendukung dua role: admin dan user.
- Memiliki autentikasi login dan pendaftaran anggota.
- Menyediakan CRUD buku, CRUD anggota, dan CRUD transaksi.
- Menyediakan pencarian buku dan transaksi.
- Menyediakan file database `.sql` untuk kebutuhan pengumpulan.

Pengembangan lanjutan yang masih bisa ditambahkan:

- denda keterlambatan,
- unggah cover buku,
- cetak laporan transaksi,
- notifikasi jatuh tempo.
