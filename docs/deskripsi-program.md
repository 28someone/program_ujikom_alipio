# Deskripsi Program

Aplikasi ini adalah sistem perpustakaan sekolah digital berbasis web menggunakan Laravel. Sistem berjalan di `localhost` dan dirancang untuk dua jenis pengguna:

- `Admin`: login, memilih menu, CRUD data buku, CRUD kelola anggota, CRUD transaksi peminjaman dan pengembalian.
- `User`: daftar, login, memilih menu, mencari buku, melakukan peminjaman buku, dan melakukan pengembalian buku.

Fitur utama yang disesuaikan dengan soal:

- Login admin dan user.
- Registrasi anggota.
- Pemilihan menu sesuai role.
- CRUD transaksi.
- CRUD data buku oleh admin.
- CRUD kelola anggota oleh admin.
- Pencarian untuk kondisi tertentu.
- Database MySQL lengkap dengan file `.sql`.
